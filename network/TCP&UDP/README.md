# TCP & UDP

## TCP
 * 연결형 서비스를 제공하는 전송계층 프로토콜
 * 높은 신뢰성을 보장
 * 연결 설정(3 way-handShake)와 해제(4 way-handShake)
 * 데이터 흐름 제어 및 혼잡 제어(네트워크 내 패킷 수가 과도하게 증가되는 경우)
 
## UDP
 * 비연결형 서비스를 제공하는 전송계층 프로토콜
 * 정보를 보내는쪽과 받는쪽이 연결된후에 정보를 전송하는 방법이 아닌 보내는 쪽이 일방적으로 정보를 보내는 방식
 * 보내는 쪽은 받는쪽이 정보를 받았던 받지 않았던 상관없이 전송한다.
 * 패킷 오버헤드가 적어 네트워크 부하 감소
