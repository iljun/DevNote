# kotlin Basic Type

| Type | Bit width |
|   --  |   --  |
|   Double  |   64  |
|   Float   |   32  |
|   Long    |   64  |
|   Int |   32  |
|   Short   |   16  |
|   Byte    |   8   |

# String
kotlin의 String은 2가지 형태로 사용 가능하다.

```val s = "Hello, world!\n"```

```val text = """
        for ( c in "foo")
            print(c)
"""
```

트리플 쿼트로 구분해서 사용가능하다.

개행과 다른 문자들을 포함할수 있는 문자열이다.
